# HZLR Web Dashboard

This is the front-end for the HZLR platform, decoupled from visual builders. It connects to our custom Node.js/PostgreSQL backend natively.

## Structure
- React + TypeScript
- Vite
- Tailwind CSS & shadcn/ui

## Getting Started
To get started locally:

1. `npm install`
2. `npm run dev` 

## Production
Ensure that environment variables correctly target your production backend API.
To test production builds locally, use:
`npm run build`
`npm run preview`
